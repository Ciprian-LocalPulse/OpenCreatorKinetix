# Support

OpenCreatorKinetix is an open-source research project.

## How To Get Help

Use GitHub Issues for:

- bug reports
- feature requests
- protocol ideas
- documentation improvements
- reproducible examples

## Good Issue Reports

Please include:

- your Julia version
- operating system
- command you ran
- expected behavior
- actual behavior
- minimal example or protocol file

## Research Discussions

For new protocol ideas, explain:

- target creator niche
- weak skill being trained
- metric expected to improve
- exercise constraints
- what evidence would prove the protocol wrong
